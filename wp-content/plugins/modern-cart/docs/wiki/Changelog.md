# Changelog

Full version history for Modern Cart Starter for WooCommerce.

---

## Version 1.0.7 — Wednesday, 4th February 2026

- **Fix:** Fixed a tax calculation issue in the cart total display, ensuring accurate tax calculation and display for WooCommerce carts.
- **Fix:** Fixed an issue where the cart appeared empty when users navigated back using the browser's back button by properly refreshing the cart state.

---

## Version 1.0.6 — Tuesday, 13th January 2026

- **Fix:** Fixed an issue with the TranslatePress plugin where unwanted HTML was appearing in product names in the cart.
- **Fix:** Fixed an issue with AJAX add-to-cart functionality by ensuring the form submits correctly when the add-to-cart button is clicked, improving the overall cart experience.

---

## Version 1.0.5 — Tuesday, 30th December 2025

- **New:** Added an onboarding wizard to provide a smooth onboarding experience and enable quick, guided setup.

---

## Version 1.0.4 — Monday, 24th November 2025

- **Fix:** Fixed a compatibility issue affecting WooCommerce's Coming Soon mode when the "Apply to store pages only" option was active.

---

## Version 1.0.3 — Thursday, 20th November 2025

- **Improvement:** Added success message to the free shipping progress bar.
- **Improvement:** Plugin now detects maintenance mode to stay inactive during maintenance.
- **Improvement:** Added option to auto-open the cart drawer when a product is added without AJAX.
- **Fix:** Improved translation support for sale percentages and cart item text.
- **Fix:** Corrected "added to cart" notification display by improving AJAX handling.

---

## Version 1.0.2 — Monday, 3rd November 2025

- **Improvement:** Improved AJAX handling in Modern Cart with cache-control to ensure always up-to-date cart data.
- **Fix:** Corrected shipping calculation errors for accurate totals when shipping settings change.
- **Fix:** Resolved Astra cart icon compatibility by refining custom trigger selector handling.
- **Fix:** Improved coupon field reliability through better state management for smoother user experience.
- **Fix:** Fixed Firefox CSS issues in the popup cart for consistent cross-browser display.

---

## Version 1.0.1 — Monday, 29th September 2025

- **New:** Added cart header customization options in Modern Cart > Styling Tab > Cart Header, allowing greater control over the appearance of your cart header.
- **Fix:** Fixed issue with tax HTML renderer not displaying correctly in the cart.
- **Fix:** Compatibility bug fixes with Modern Cart Pro.

---

## Version 1.0.0 — Tuesday, 9th September 2025

- Initial release of Modern Cart Starter for WooCommerce.
